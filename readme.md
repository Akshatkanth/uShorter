# uShorter
https://roadmap.sh/projects/url-shortening-service
uShorter is a simple **URL shortening service** built with a React frontend and a Node.js + Express backend.
It allows users to create short links, redirect to original URLs, and view usage statistics.

---

## Features

* Create short URLs
* Redirect to original URLs
* View access statistics
* Update and delete short URLs

---

## Tech Stack

* React (Vite)
* Node.js, Express
* MongoDB Atlas
* Mongoose
* NanoID
* Deployed on Netlify (frontend) and Render (backend)

---

## API Overview

* `POST /shorten` – Create short URL
* `GET /:shortCode` – Redirect
* `GET /shorten/:shortCode/stats` – Get statistics
* `PUT /shorten/:shortCode` – Update URL
* `DELETE /shorten/:shortCode` – Delete URL

---

## Deployment

* Frontend: Netlify
* Backend: Render
* Database: MongoDB Atlas
